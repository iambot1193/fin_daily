O Findaily é uma aplicação Full Stack que tem como objetivo ajudar usuários a organizar sua vida financeira por meio de uma interface moderna e intuitiva. A plataforma permitirá gerenciar receitas, despesas, contas bancárias, cartões de crédito, faturas, parcelamentos, metas financeiras e lembretes, oferecendo uma visão completa da situação financeira do usuário.

O projeto também servirá como estudo e aplicação de boas práticas de desenvolvimento, arquitetura de software e tecnologias modernas para construção de APIs e aplicações web.

✨ Funcionalidades planejadas
👤 Autenticação de usuários (JWT)
💰 Controle de receitas e despesas
🏦 Gerenciamento de contas bancárias
💳 Cadastro de cartões de crédito
🧾 Controle de faturas e compras parceladas
📅 Agenda de vencimentos e lembretes
🎯 Metas financeiras
📊 Dashboard com indicadores financeiros
🔍 Pesquisa, filtros e paginação
📄 Documentação da API (Swagger/OpenAPI)
